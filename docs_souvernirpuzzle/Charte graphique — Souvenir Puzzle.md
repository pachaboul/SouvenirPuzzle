Charte graphique — Souvenir Puzzle
1. Identité de marque

Nom : Souvenir Puzzle
Univers : souvenir, famille, amour, mémoire, puzzle, émotion, transmission.
Positionnement : un jeu mobile familial qui transforme les photos personnelles en puzzles émotionnels.
Promesse :
“Transformez vos photos en puzzles et revivez vos souvenirs pièce par pièce.”

Souvenir Puzzle doit donner une impression de chaleur, confiance, intimité, sécurité et élégance familiale.

2. Logo

Le logo principal représente :

un cœur : amour, famille, émotion ;
deux formes de pièces de puzzle : jeu, reconstruction, souvenir ;
des profils parent/enfant : famille, lien affectif, transmission ;
un petit cœur central : mémoire personnelle et attachement.
Version principale

Logo en bleu nuit + or.

Utilisation recommandée :

splash screen ;
icône d’application ;
page d’accueil ;
présentation officielle ;
site web ;
Play Store / App Store.
Versions secondaires
Version monochrome noire
Pour documents, impression simple, watermark, facture, dossier de présentation.
Version monochrome blanche
Pour fond sombre, vidéo, bannière, footer, écran de chargement sombre.
Version outline / line art
Pour stickers, favicon, petites tailles, éléments décoratifs.
Version icône seule
Sans texte, utilisée comme app icon ou avatar.

6. Style visuel

Le style général doit être :

premium mais chaleureux ;
familial mais pas enfantin ;
émotionnel mais propre ;
moderne mais simple ;
calme, non agressif.

À éviter :

couleurs trop flashy ;
trop de néon ;
interface trop gaming agressive ;
effets trop chargés ;
typographie enfantine excessive.

13. Règles d’utilisation du logo
À faire
garder le logo lisible ;
laisser de l’espace autour ;
utiliser la version couleur sur fond clair ;
utiliser la version blanche sur fond sombre ;
utiliser l’icône seule pour app icon.
À éviter
étirer le logo ;
changer les proportions ;
ajouter trop d’effets ;
mettre sur un fond trop chargé ;
utiliser des couleurs hors charte ;
réduire trop petit si les profils ne sont plus lisibles.
14. App Icon

Icône recommandée :

fond Bleu Nuit ;

silhouette claire ou négative ;
coins arrondis ;
pas de texte.

L’icône doit être lisible même en petite taille.
Pour l’App Store / Play Store, il vaut mieux utiliser logo symbole seul, sans “Souvenir Puzzle”.

16. Direction recommandée

Pour Souvenir Puzzle, la meilleure direction est :

Logo principal : cœur puzzle parent/enfant en bleu nuit + or.
Interface : crème clair + bleu nuit + accents or.
Style : premium familial, émotionnel, calme.

Cette direction parle directement parce qu’elle combine les trois messages essentiels :

Puzzle : forme des pièces.
Souvenir : cœur et émotion.
Famille : silhouettes parent/enfant.

Résumé de marque :

Souvenir Puzzle est une application douce, familiale et premium qui transforme les photos personnelles en puzzles émotionnels, tout en respectant la vie privée de l’utilisateur.
